# PlatinumRX_Assignment

## Approach & Methodology

### 1. SQL Proficiency

I developed two separate relational schemas using SQLite syntax.

- Hotel System: Focused on join operations and window functions (RANK, ROW_NUMBER) to identify chronological events (last booking) and performance metrics (top items).

- Clinic System: Focused on financial aggregation, calculating monthly profitability by joining revenue and expense tables on dates.

### 2. Spreadsheet Proficiency

- Data Integration: Used INDEX-MATCH for flexible data retrieval between sheets.

- Time Analysis: Applied INT() and HOUR() functions to timestamps to isolate specific temporal dimensions for efficiency reporting.

### 3. Python Proficiency

- Logic: Utilized modular arithmetic (// and %) for the time converter to ensure accurate unit splitting.

- Strings: Implemented a membership-check loop to maintain the original order of characters while filtering duplicates.

## Folder Structure

- /SQL: Database setup scripts and analytical queries.

- /Spreadsheets: Documentation of logic and formulas used in the Excel workbook.

- /Python: Executable scripts for the logic tasks.
